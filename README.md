# 下班助手

一个简洁优雅的 macOS 菜单栏应用，帮助你追踪工作时间和下班倒计时。

![macOS](https://img.shields.io/badge/macOS-15.7+-blue.svg)
![Swift](https://img.shields.io/badge/Swift-5.9+-orange.svg)
![License](https://img.shields.io/badge/license-MIT-green.svg)

## 功能特性

### 核心功能
- ⏰ **实时倒计时** - 精确到秒的下班倒计时显示，支持工作状态智能识别
- 📅 **周末倒计时** - 显示距离周末还有多久，自动识别工作日
- 🎉 **假期倒计时** - 自动获取国家法定节假日，显示下一个假期倒计时
- 🔔 **智能提醒** - 支持下班前提醒和午休前提醒，可自定义提前时间
- 💡 **AI 关怀助手** - 在工作时间发送温馨的健康提醒（休息、喝水等）
- 🐾 **虚拟宠物助手** - 可爱的 3D 宠物陪伴，通过对话气泡发送提醒
- 🌓 **暗黑模式** - 完美适配 macOS 浅色/深色主题

### 界面特点
- 🎯 **菜单栏显示** - 倒计时直接显示在菜单栏，一目了然
- 🖱️ **点击切换** - 点击菜单栏图标即可显示/隐藏主窗口
- 🎨 **现代设计** - 简洁美观的界面设计，采用玻璃态效果
- 🚫 **无 Dock 图标** - 不占用 Dock 空间，保持桌面整洁
- 🎭 **精美动画** - 流畅的过渡动画和交互反馈

### 虚拟宠物功能
- 🐱 **多种宠物类型** - 支持猫、狗、兔子等多种 3D 宠物模型
- 💬 **对话气泡** - 现代化的对话气泡设计，支持深色模式
- 📍 **随机位置** - 每次出现在屏幕随机位置，增加趣味性
- ⏱️ **自动隐藏** - 可配置自动隐藏延迟时间
- 🎨 **透明度调节** - 可自定义窗口透明度
- 🖱️ **自由拖动** - 支持拖动宠物到任意位置

### 自定义设置
- ⚙️ **工作时间配置** - 使用下拉选择器自定义上班、下班时间
- 🍱 **午休时间设置** - 配置午休开始时间和时长（默认 90 分钟）
- 🔔 **通知设置** - 自定义下班提醒和午休提醒的提前时间
- 🤖 **AI 关怀设置** - 配置 AI 服务格式、API Key、模型等
- 🐾 **宠物设置** - 选择宠物类型、配置自动隐藏时间和透明度

## 系统要求

- macOS 15.7 或更高版本
- Apple Silicon 或 Intel 处理器

## 安装

1. 下载最新版本的 `.app` 文件
2. 将应用拖入"应用程序"文件夹
3. 首次打开时，可能需要在"系统设置 > 隐私与安全性"中允许运行

## 使用说明

### 首次使用

1. 启动应用后，菜单栏会显示倒计时时间
2. 点击菜单栏图标打开主窗口
3. 点击主窗口中的"设置"按钮配置工作时间

### 配置工作时间

1. 在主窗口点击"设置"按钮
2. 设置上班时间（如 09:00）
3. 设置下班时间（如 18:00）
4. 设置午休开始时间和时长
5. 设置会自动保存

### 配置通知提醒

1. 打开设置窗口
2. 启用"启用下班提醒"开关
3. 设置提前提醒时间（默认 30 分钟）
4. 启用"启用午休提醒"开关（可选）
5. 设置午休提前提醒时间（默认 10 分钟）

### 配置 AI 关怀助手

1. 打开设置窗口
2. 启用"启用 AI 关怀"开关
3. 选择服务格式（OpenAI 格式或 Anthropic 格式）
4. 输入 API Key
5. 配置模型名称（可选）
6. 配置基础 URL（可选）
7. 设置提醒间隔（默认 120 分钟）
8. 点击"测试配置"验证连接

### 配置虚拟宠物

1. 打开设置窗口
2. 启用"启用虚拟宠物"开关
3. 选择宠物类型（猫、狗、兔子等）
4. 设置自动隐藏延迟（默认 10 秒）
5. 调整窗口透明度（0.5-1.0）
6. 点击"测试宠物弹窗"预览效果

**注意事项：**
- 启用虚拟宠物后，所有提醒将通过宠物对话气泡显示
- 禁用虚拟宠物后，将使用系统通知
- 宠物窗口可以自由拖动到任意位置
- 每次出现位置随机，增加趣味性

## 技术栈

- **语言**: Swift 5.9+
- **框架**: SwiftUI
- **架构**: MVVM
- **核心技术**:
  - UserNotifications - 系统通知功能
  - NSStatusBar - 菜单栏集成
  - URLSession - 网络请求（假期数据、AI API）
  - SceneKit - 3D 宠物模型渲染
  - Combine - 响应式编程
- **第三方服务**:
  - 中国法定节假日 API
  - OpenAI 兼容 API（智谱 AI、OpenAI 等）
  - Anthropic Claude API

- Xcode 15.0 或更高版本
